# 🎓 Student Performance Prediction using Machine Learning

## 📌 Overview

This project predicts student academic performance using machine learning techniques.
The goal is to analyze student-related factors and predict whether a student is likely to perform well academically.

The project demonstrates data preprocessing, exploratory data analysis, machine learning model training, and model evaluation.

---

## 📊 Dataset

The dataset contains information about student characteristics and exam scores.

Features include:

* Gender
* Parental level of education
* Lunch type
* Test preparation course
* Math score
* Reading score
* Writing score

Dataset file: `StudentsPerformance.csv`

---

## 🤖 Machine Learning Models Used

The following algorithms were implemented and compared:

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* Neural Network

Each model was evaluated to determine which provides the best prediction accuracy.
